# vlp-surrogate
Data-driven estimation of vertical lift performance (VLP) curves for accurate tubing head pressure (THP) simulation.


## How to run:

For information regarding how to run the script run:

```shell
python src/main.py -h
```

Example run with GA enabled:

```shell
python src/main.py --file-path H-12.xlsx --file-type p --output output.vlp --model ga
```
